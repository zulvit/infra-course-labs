# Документация для Codex

Этот каталог собран так, чтобы новый исполнитель мог быстро понять структуру проекта и начать работу без поиска промптов по корню репозитория.

## Содержимое

- `prompts/` — готовые текстовые инструкции для Codex;
- `guides/` — короткие технические памятки;
- `status/` — сводные документы по текущему состоянию выполненных работ.

## Рекомендуемый порядок использования

1. Прочитать `prompts/autonomous-lab-execution.md`, если нужно продолжать выполнение или дорабатывать работы.
2. Использовать `prompts/otchet-compose-report-generation.md`, если задача связана с пересборкой или обновлением DOCX-отчётов.
3. Открыть `guides/reports-docx.md`, если нужен короткий справочник по массовой генерации отчётов.
4. Проверить `../status/completed-labs-summary.md`, если нужен общий контекст по уже выполненным лабораторным.
